addOrderingAndLimit

protected final T addOrderingAndLimit<T extends AbstractSQLQuery<?, ?>>(T query, OrderBy ordering, Integer limit)

Appends the given query with the ordering and limit, if they are provided.

The limit value is applied if and only if a meaningful ordering directive is provided.

In case the ordering directive is set, and the limit value is zero or less, an IllegalStateException is thrown.

If only the limit value is supplied, and the ordering directive is either null, or is equal to OrderBy.getDefaultInstance(), an IllegalStateException is thrown as well.

Return

the same query instance

Parameters

query

query to configure

ordering

ordering directive, null, if not set by end-users

limit

the maximum number of records to return, null, if not specified

<T>

the type of the query
