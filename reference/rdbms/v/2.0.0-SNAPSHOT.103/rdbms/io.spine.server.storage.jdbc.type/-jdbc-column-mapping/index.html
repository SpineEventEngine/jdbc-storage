JdbcColumnMapping

@Experimental
@SPI
open class JdbcColumnMapping : AbstractColumnMapping<Any>

Scheme of relations between Java types of the values stored in record columns, and storage-specific types.

Also defines the RDBMS-level types for the table columns.

Users may extend this type to add own custom mapping for some of the stored types.
