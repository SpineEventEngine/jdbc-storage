Package io.spine.server.storage.jdbc.operation

Class WriteOne

  • All Implemented Interfaces:
    
public class WriteOne<I, R extends Message>
extends Operation<I, R>

    Writes a single record to the database.

    This is a generic version of the operation, which performs an additional SELECT lookup before either executing an UPDATE query or an INSERT query.
