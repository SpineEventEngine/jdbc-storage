    • Constructor Detail

      • JdbcTableSpec

        JdbcTableSpec(String tableName, RecordSpec<I, R> recordSpec, JdbcColumnMapping mapping)

        Creates a new table specification, also setting a custom name for the table.

        It is a responsibility of callers to select the table name which is both unique and compatible with the requirements of the underlying database engine.

        Parameters:
        tableName - the name to use for the table
        recordSpec - the original specification of the stored record
        mapping - the column mapping to use
