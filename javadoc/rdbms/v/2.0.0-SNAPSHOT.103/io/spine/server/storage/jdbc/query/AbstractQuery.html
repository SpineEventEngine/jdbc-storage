      • mySqlFactory

         static MySQLQueryFactory mySqlFactory(DataSourceWrapper dataSource)

        Creates a new MySQL-specific query factory.

        Transactional settings of the returned factory are the same as for the default query factory.

        The created factory optimizes the inserts and updates into DB tables by leveraging INSERT ... ON DUPLICATE KEY UPDATE ... queries instead of SELECT ... with the consecutive INSERT/UPDATE.

        Parameters:
        dataSource - the data source to produce connections
        Returns:

        a new query factory
