Package io.spine.server.storage.jdbc.operation

Class OperationFactory

  • All Implemented Interfaces:
    @SPI() 
public class OperationFactory

    A factory of Operations.

    Descendants may extend this type in order to customize the execution of certain operations performed over the underlying database.

    End-users may choose to use their own operation implementations by extending this type, returning appropriate operation implementations, and using this custom operation factory to build a JdbcStorageFactory.

    Also, all query types are made public and marked as SPI elements, since they may also be involved into customizing the actual SQL queries.
